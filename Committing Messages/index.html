This part is mostly about how to commit messages


To keep in mind on how to commit messages is to..

        "Describe the problem your commit solves and how."

There's only 72 character limit in GitHub

so shoot for 50 characters but consider 72 the hard limit.

try using this in a sentence.

Examples:

  • If applied, this commit will refactor subsystem X for readability
  • If applied, this commit will update getting started documentation
  • If applied, this commit will remove deprecated methods
  • If applied, this commit will release version 1.0.0
  • If applied, this commit will merge pull request #123 from user/branch

Just remove the the "If applied, this commit will" then the remaining words are your commit messages.
